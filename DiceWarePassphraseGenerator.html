Welcome to the Dice Ware Passphrase Generator.

This website generates a secure Passphrase through the DiceWare algorithm.
A dice is thrown a number of times, this results in a number that can be matched to a word in a DiceWare Wordlist.
More information in the algorithm and the WordLists that can be used are available in the How It Works page.
The more words the passphrase has, the more secure it is. However, it is also harder to remember.
Try to find a balance between security and memorability.

Steps:

1. Input a .txt file with the Wordlist you want to use. Remember it must follow the DiceWare Algorithm.

2. Enter how many dice throws your Wordlist requires. The default Wordlists provided use 5 Dice throws.

3. Enter how long you want the passphrase to be. In other words, the number of words in the passphrase.

4. Click on the Generate Button and see the results.
