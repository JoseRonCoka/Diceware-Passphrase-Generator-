Dice Ware Passphrase Generator

Acronym Passphrase Generator How it Works

Welcome to the Dice Ware Passphrase Generator.

This website generates a secure Passphrase through the DiceWare algorithm.
To select a word a dice is thrown a number of times, depending on the results it got a number is picked from a DiceWare Wordlist. More information in the algorithm and the WordLists that can be used are available in the manual. Besides generating a purely random passphrase it can also generate a Acronym passphrase. Meaning you can enter a meaningful word to you, and the Passphrase generated will be made up by words that start with each of the characters that the word you enter contain. This should make it easier to remember.

Steps:

1. Input a .txt file with the Wordlist you want to use. Remember it must follow the DiceWare Algorithm.

2. Enter how many dice throws your Wordlist requires. The default Wordlists provided use 5 Dice throws.

3. Enter how many words you want the passphrase to have. In other words, the number of words in the passphrase.

4. Leave empty or enter the meaningful word to form the Acronym Passphrase. IT MUST HAVE AS MANY LETTERS AS THE PASSSPHRASE LENGTH YOU ENTERED IN THE PREVIOUS STEP.

5. Click on the Generate Button and see the results.

TO RUN AGAIN WITH SAME WORDLIST: Repeat Steps 2 through 5.

TO RUN AGAIN WITH DIFFERENT WORDLIST: Reload the page and repeat steps 1 through 5.
